The repository contains software for

- Channel registration

- Analysis of blinking statistics

- Ripley's K analysis of experimental data and simulations



The following scripts allow to run the software on exemplary data:

run_channelRegistration.m

run_analyseBlinkStat.m

run_ripleyAnalysis.m



Documentation of the code may be found in:

documentation.pdf



The repository contains the following folders:

- analyseBlinkStat: files for blinking analysis

- channelRegistration: files for channel registration

- helperfunctions: helper files needed for the software

- ripleyAnalysis: files for Ripley's K analysis

- testdata: exemplary data for running the code

- test_results: results for test data
